# Code-Editor-Live
A browser-based HTML, CSS, and JavaScript editor built for quick experiments, learning, and live preview—no setup required.

A lightweight browser-based code editor for HTML, CSS, and JavaScript with live preview.

## Features
- Live HTML/CSS/JS preview
- Multiple editor themes
- Font zoom controls (A− / A+)
- Auto-run preview
- Save and load projects as JSON
- Runs completely inside the browser

## Tech Stack
- HTML
- CSS
- JavaScript
- Ace Editor
